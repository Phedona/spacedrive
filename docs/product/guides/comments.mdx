# Comments

<Notice
	type="note"
	title="WIP"
	text="This feature is not available yet, please check our [roadmap](/roadmap)."
/>

Comments allow you to annotate files, on the file itself or within it at a given timestamp and/or coordinate. If a file is shared, other users it is shared with will be able to see comments and reply depending on the permissions. Comments are syncronized in realtime between Nodes.
